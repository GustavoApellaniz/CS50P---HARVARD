# CS50P---HARVARD
Solutions to the exercises from Harvard’s CS50P (Introduction to Programming with Python)


This repository contains my solutions to the exercises and problem sets from **CS50P – Introduction to Programming with Python**, offered by Harvard University.

The goal of this repository is to document my progress throughout the course while practicing Python fundamentals, problem solving, and programming logic.

---

## Course Topics

The course covers several important programming concepts, including:

- Functions
- Conditionals
- Loops
- Exceptions
- File I/O
- Libraries
- Unit Testing
- Object-Oriented Programming
- Regular Expressions

---

## Repository Structure

Each folder corresponds to a week or problem set from the course.
